Coding Theory-Some Bounds

еҸ‘иЎЁдәҺ 2019-12-26 жӣҙж–°дәҺ 2019-12-30 еҲҶзұ»дәҺ Coding Theory йҳ…иҜ»ж¬Ўж•°пјҡ иҜ„и®әж•°пјҡ

жң¬иҠӮдё»иҰҒд»Ӣз»Қе…ідәҺзј–з Ғзҡ„дёҖдәӣдёҠдёӢз•ҢгҖӮ

The Main Coding Theory Problem

е®ҡд№ү$A_q(n, d)$дёәй•ҝеәҰдёә$n$, жңҖе°Ҹи·қзҰ»жҳҜ$d$зҡ„зј–з Ғ, sizeжңҖеӨ§иғҪжҳҜеӨҡе°‘гҖӮ$B_q(n, d)$дёәй•ҝеәҰдёә$n$, жңҖе°Ҹи·қзҰ»жҳҜ$d$зҡ„зәҝжҖ§з Ғ, sizeжңҖеӨ§иғҪжҳҜеӨҡе°‘гҖӮ зЎ®е®ҡе®ғд»¬зҡ„еҖјжҲ–иҖ…дёҠдёӢз•ҢжҲҗдёәThe Main Coding Theory Problem.

йҰ–е…ҲжҳҜеҮ дёӘйқһеёёtrivialзҡ„bound:

  1. $B_q(n, d) \leq A_q(n, d) \leq q^n$.
  2. $B_q(n, 1) = A_q(n, 1) = 1$.
  3. $B_q(n, n) = A_q(n, n) = n$.

еүҚдёӨдёӘйқһеёёжҳҫз„¶, жңҖеҗҺдёҖдёӘеӣ дёә$d=n$, йӮЈд№ҲжҜҸдёӘcodewordзҡ„д»»ж„ҸдҪҚдә’дёҚзӣёеҗҢгҖӮ еҸӘиҖғиҷ‘з¬¬дёҖдҪҚдә’дёҚзӣёеҗҢпјҢжңҖеӨҡе°ұеҸӘиғҪйҖү$n$дёӘcodewordгҖӮеӣ жӯӨдёҠз•ҢжҳҜ$n$. иҖҢ$\{(a, a, \cdots, a) \mid a \in F_q\}$е°ұеҸҜд»ҘиҫҫеҲ°$n$.

дёӢйқўеј•е…ҘдёҖдёӘextended codeзҡ„жҰӮеҝө:

е°ұжҳҜеңЁжңҖеҗҺеўһеҠ дёҖдҪҚ, жҳҜеүҚйқўзҡ„жүҖжңүдҪҚзҡ„е’ҢеҸ–иҙҹгҖӮ

Theorem: еҒҮи®ҫ$d$жҳҜдёҖдёӘеҘҮж•°гҖӮ

  1. еӯҳеңЁ$(n, M, d)_2$-codeеҪ“дё”д»…еҪ“еӯҳеңЁ$(n+1, M, d+1)_2$-code. $A_2(n, d) = A_2(n+1,d+1)$.
    иҜҒжҳҺпјҡ
    еҒҮи®ҫ$C$жҳҜ$(n, M, d)_2$-code, иҖғиҷ‘е®ғзҡ„extended code $\overline{C}$. жҳҫз„¶$d \leq d(\overline{C}) \leq d+1$. иҖғиҷ‘д»»ж„Ҹ$u,v \in C, d(u, v) = d$. з”ұдәҺ$d$жҳҜеҘҮж•°, $\overline{u}, \overline{v}$йҷӨдәҶ$u,v$дёҚеҗҢзҡ„дҪҚзҪ®дёҚеҗҢ, жңҖеҗҺдёҖдҪҚжү©еұ•дҪҚд№ҹдёҚеҗҢгҖӮеӣ жӯӨ$d(\overline{u}, \overline{v}) = d+1$. д»ҺиҖҢиҜҙжҳҺ$\overline{C}$жҳҜ$(n+1, M, d+1)_2$-code.
    еҸҚиҝҮжқҘпјҢеҰӮжһң$C$жҳҜ$(n+1, M, d+1)_2$-code, йӮЈд№ҲеӯҳеңЁ$u,v \in C, d(u, v)=d+1$. жүҫеҲ°$u,v$дёҚзӣёеҗҢзҡ„жҹҗдёҖдҪҚпјҢ жҠҠ$C$дёӯжүҖжңүcodewordеҲ еҺ»иҝҷдёҖдҪҚ, еҫ—еҲ°ж–°зҡ„з Ғ$CвҖҷ$гҖӮ еӣ дёәеҸӘеҲ еҺ»дәҶдёҖдҪҚпјҢйӮЈд№Ҳи·қзҰ»жңҖеӨҡеҮҸе°‘1пјҢеӣ жӯӨ$d(CвҖҷ) \geq d$, иҖҢ$d(uвҖҷ, vвҖҷ)=d$, жүҖд»Ҙ$d(CвҖҷ) = d$.
  2. еӯҳеңЁ$[n, M,d]_2$-codeеҪ“дё”д»…еҪ“еӯҳеңЁ$[n+1, M, d+1]$-code. $B_2(n, d) = B_2(n+1,d+1)$.
    иҜҒжҳҺпјҡ
    еҰӮжһң$C$жҳҜlinear code, йӮЈд№Ҳ$\overline{C}$д№ҹжҳҜпјҢеҝ…иҰҒжҖ§еҫ—иҜҒгҖӮ
    еҰӮжһң$C$жҳҜ$[n+1, M, d+1]_2$-code, йӮЈд№ҲеӯҳеңЁ$x \in C, wt(x) = d+1$. йҖүжӢ©$x$йқһ0зҡ„жҹҗдёҖдҪҚ, жҠҠ$C$дёӯжүҖжңүcodewordеҲ еҺ»иҝҷдёҖдҪҚ, еҫ—еҲ°ж–°зҡ„з Ғ$CвҖҷ$, жҳҫз„¶$CвҖҷ$д№ҹжҳҜlinear code. д№ӢеҗҺеҗҢдёҠдёҖжқЎеҺ»жҺүжҹҗдёҖдҪҚзҡ„и®әиҜҒгҖӮ

Some Lower Bounds

жң¬иҠӮдё»иҰҒи®Ёи®әе…ідәҺ$A_q(n, d), B_q(n, d)$зҡ„дёҖдәӣдёӢз•ҢгҖӮ

Sphere-Covering Bound

е®ҡд№ү$S(u, r):= \{v \in C \mid d(u,v) \leq r \}$. жҳҫз„¶$|S(u, r)| = \sum_{i=0}^{r} \binom{n}{i}(q-1)^i$. д№ҹе°ұжҳҜд»Ҙ$u$дёәеңҶеҝғпјҢ$r$дёәеҚҠеҫ„зҡ„зҗғгҖӮ
иҖғиҷ‘иҫҫеҲ°жңҖдјҳеӨ§е°Ҹ$A_q(n,d)$зҡ„з Ғ$C$, $|C| = A_q(n, d)$, жңү$\bigcup_{u \in C}{S(u, d)} = F_q^n$. еӣ дёәеҰӮжһңиҝҷдәӣзҗғе№¶иө·жқҘдёҚжҳҜж•ҙдёӘз©әй—ҙпјҢйӮЈд№ҲеӯҳеңЁдёҖдёӘзӮ№дёҚеңЁд»»ж„ҸдёҖдёӘзҗғеҶ…пјҢиҝҷдёӘзӮ№е’Ң$C$еҶ…зҡ„д»»ж„Ҹcodewordи·қзҰ»йғҪи¶…иҝҮ$d$, йӮЈд№ҲиҝҷдёӘзӮ№еҠ е…Ҙ$C$еҸҜд»Ҙеҫ—еҲ°жӣҙеӨ§зҡ„з ҒгҖӮжүҖд»Ҙжңү

GilbertвҖ“Varshamov Bound

Theorem:
еҰӮжһң

йӮЈд№ҲеӯҳеңЁ$[n, k]$ linear code $C$, $d(C) \geq d$.
иҜҒжҳҺ:
иҖғиҷ‘жһ„йҖ дёҖдёӘ$(n-k) \times n$зҡ„вҖқcheck matrixвҖқ $H$, иҝҷйҮҢеҠ еј•еҸ·жҳҜеӣ дёәе№¶дёҚиҰҒжұӮ$H$зҡ„иЎҢжҳҜзәҝжҖ§ж— е…ізҡ„, дҪҶжҳҜд»»ж„Ҹ$d-1$еҲ—жҳҜзәҝжҖ§ж— е…ізҡ„гҖӮйӮЈд№Ҳ$H$зҡ„null spaceеҜ№еә”зҡ„е°ұжҳҜдёҖдёӘз»ҙж•°иҮіе°‘жҳҜ$k$, и·қзҰ»иҮіе°‘жҳҜ$d$зҡ„зәҝжҖ§з ҒгҖӮд»»еҸ–generator matrix $G$зҡ„$k$иЎҢе°ұеҸҜд»Ҙеҫ—еҲ°з»ҙж•°жҒ°еҘҪжҳҜ$k$, и·қзҰ»иҮіе°‘жҳҜ$d$зҡ„зәҝжҖ§з ҒгҖӮ
дёӢйқўиҖғиҷ‘дёҖеҲ—дёҖеҲ—жһ„йҖ иҝҷж ·зҡ„$H$:

  1. йҡҸдҫҝйҖүдёҖдёӘ$u_1 \in F_q^{n-k}$, дҪңдёәз¬¬1еҲ—гҖӮ
  2. еҜ№$2\leq i \leq n$, йҖүеҸ–$u_i$дҪңдёәз¬¬$i$еҲ—, иҰҒжұӮ$u_i$дёҚиғҪдҪҝ$\{u_1, u_2, \cdots, u_{i-1}\}$дёӯд»»ж„Ҹ$d-2$дёӘзҡ„зәҝжҖ§з»„еҗҲгҖӮ
    жҳҫз„¶еҰӮжһңжһ„йҖ йЎәеҲ©, жҲ‘д»¬е°ұжүҫеҲ°дәҶз¬ҰеҗҲиҰҒжұӮзҡ„$H$, дҪҶжҳҜжҲ‘д»¬иҝҳеҫ—иҜҒжҳҺз¬¬дәҢжӯҘдёӯдёҖе®ҡеӯҳеңЁдёҖдёӘз¬ҰеҗҲиҰҒжұӮзҡ„$u_i$.
    еӣ дёә$\{u_1, u_2, \cdots, u_{i-1}\}$дёӯд»»ж„Ҹ$d-2$дёӘзҡ„зәҝжҖ§з»„еҗҲжһ„жҲҗзҡ„з©әй—ҙеӨ§е°ҸдёҚи¶…иҝҮжүҖд»ҘжҜҸж¬ЎжҖ»иғҪжүҫеҲ°з¬ҰеҗҲиҰҒжұӮзҡ„$u_i$.
