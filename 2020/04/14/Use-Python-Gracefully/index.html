如何优雅地使用 Python

2020-04-14 Knowledge Learning 阅读量

导语

每当我在用 python 做一些 task 的时候，对这门语言总有强烈的距离感。

  • 很多基础的函数不会调用，要现场百度起来。
  • 更糟糕的是，没有掌握 Python 灵巧的编程思想，还是一味用 C++ 的思路去写代码。

于是我就开了这篇博客，记录一些反复用到的技巧和方法，培养 Python 的思维习惯。

简化代码的函数

  • any(<list>)，all(<list>)

  • 用 enumerate() 来带序号遍历 list

  • 取 dict 里的元素最大值：max(vote, key = vote.get)

  • 字符串相关的函数

  • 正则表达式

奇技淫巧

  • namedtuple 具名元组：在元组的基础上，支持用属性名去访问元素。

  • partial：灵活设置函数的默认参数，使其变成新函数。

常见的私有方法

类或函数的方法 效果
__doc__ 描述信息，即定义开头长注释里的字符串；可用于模块、函数、类。
__new__，__init__ 前者用于类对象的创建（静态方法），后者用于对象的初始化（实例方法）。前者必须要给出返回值（对象），作为后者的第一个参数。
__del__ 类的析构函数
__module__，__class__ 前者表示对象所在模块，后者表示对象所属的类
__dict__ 类或对象中的所有成员（dict 形式）
__str__ 类需要转为字符串时（比如打印该对象）触发，类似于 Java 的 toString
__call__ 当对象后面接 () 后会触发
__getitem__ 当对象后面接 [] 后会触发
__setitem__，__delitem__ 连同上面的 __getitem__，使类像 dict 一样使用
__iter__， __next__ 使该类变得可迭代，可以用 for ... in ... 遍历

Numpy 入门

  • numpy 的初始化

  • 切片和索引

    • 索引规则和普通数组差不多，用 : 和 , 分隔。... 表示这一维度不要求。

    • 也可以由数组下标进行索引。一共传进去 dim 个数组，内部性状决定了结果形状。

    • 布尔索引和花式索引。

Conda 和 Pip

需求 指令
查看当前所有环境 conda info --env
新建一个环境 conda create --name <name> python=3.5
激活环境 conda activate<name>
关闭环境 deactivate <name>
删除环境 conda remove -n python35 --all
pip 换源 pip install -i <url> <package_name>
