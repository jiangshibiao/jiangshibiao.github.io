Game-Collect-In-Seer

Created2022-11-20|Updated2022-11-24|Game
|Word count:3.3k|Reading time:10min|Post View:

在这篇文章里，我根据自己的喜好整理出了周更游戏《赛尔号》在一年半里的每周活动。筛选规则如下：

  • 必须要有决策性和趣味性的游戏，单纯的战斗或点击过剧情会被略过。
  • 绝大部分都是思维性的游戏，一小部分会涉及操作，因为本人是操作渣。

我赛的游戏风格为：基本都是技术+运气相结合，比重平均六四开，无脑操作一般也能拿满奖励。

每个游戏的标题由更新日期和官方命名组成，以任意一个作为关键词都能在 B 站搜到介绍或玩法。

  • 游戏中的攻略都是自己捣鼓出来的，挑了些分数比较高的结果截图，如有更好的策略欢迎提出。
  • 游戏规则的介绍图我当时没怎么收集，大多是现场去 B 站搜的，所以右上角会带有水印，侵删。

2021-4-9/2022-4-8 重建枫曳镇

游戏规则介绍：4×44 \times 44×4 的空地，底下会随机出两个材料，使用其中一个后另一个会重新随机。材料共分为四种：玻璃、砖块、枫曳和石头。特定材料组成特定形状（方向必须一致，不能旋转）后可升级成一个特定的房型，升级位置可选在材料覆盖的格子中的任意一格。每一种房型会触发一定的计分规则，当空地被填满时游戏结束。

个人攻略和感想：优先拼房舍和谷仓，一般的 pattern 是谷仓在 (2,1)(2,1)(2,1) 然后周围盖房子，房舍边上再慢慢堆集市。卡材料的话可以考虑在右下角的空地堆广场、教堂或水井。很难一路顺风顺水，经常面对卡材料的窘境。

2021-4-9/2022-4-8 耕种庆典

游戏规则介绍：初始金币 200020002000。一共有 999 种蔬菜，分为批发市场和农贸大卖场两个场地。前者每次会刷新 555 种蔬菜可以购买，其中 333 种单买，还有 222 种需要捆绑卖（价格会打一个小折扣），只要资金足够可以无限购买；后者每次会刷新 444 种蔬菜，可卖出有库存的蔬菜。每种蔬菜的卖出价格略高于买入价格。此外，每过若干回合会更新一个场地 buff，会导致买入或卖出的价格有一些浮动。游戏持续 151515 回合，每一次场地转换或刷新买入/卖出的蔬菜列表都需要消耗一个回合。要求回合结束时的金币尽量多。

个人攻略和感想：第一年的策略比较稳重：转场需要消耗回合，在批发市场优先选捆绑和折扣低的买，每次买入的总价占总资产的 1/21/21/2 或 1/31/31/3；基本花光钱时再去卖场卖菜，卖也尽量追求卖光——收盘金币稳定在 10000−10000-10000− 。第二年在逛贴吧时学到了一个神级操作：每次 all in 一种蔬菜买入/卖出，迅速切换场地，每种场地最多刷新一次。再细腻地预判一下场地 buff，每几回合就能打到一次 20000+20000+20000+。

2021-9-10/2022-10-21 封绝之阵/瓦普特号修复计划

游戏规则介绍：5×55 \times 55×5 的棋盘，玩家和电脑交替行动。电脑只有一种棋子，摆出特定的阵型时随机移除一颗玩家的棋子；玩家有风火水地四种棋子，摆出特定的阵型时会产生特殊的效果（见左下）。每回合系统会随机两个不同的棋子，玩家选一个摆放。玩家还能预知下一回合的两个棋子。当棋盘上已经没有棋子时，按场上的棋子个数结算胜者。该游戏还有一个成就系统，达成过某种局面才能解锁（见右下）。

个人攻略和感想：敌方阵型预判起来比较麻烦，我在思考时直接把游戏简化成“敌方每落下一颗棋子就有概率移除己方一颗棋子，且敌方在战场上的棋子越多，这个概率就越大”。四象封印和四象结界难记，也不方便多次触发，除了为了解锁相关成就外不推荐使用。土和风比较容易触发（横竖放比斜着放的方案更多），但会影响到自家的棋子的状态（前者会移除，后者会重随），容易打破自己的计划。实战中效果最好的是火，无条件移除敌方三个棋子并把对应位置置为不可用，不仅能直接优化目标函数，还能加快游戏进度，一举两得。还有一个小技巧：开场时优先把火放在靠中间的对角线上，因为满三个触发一次后，再在边上补一个又能触发一次。

2022-1-28 烟花试验田

游戏规则介绍：游戏分成四个关卡。每一个关卡初始时给出一块空地，有 444 种正常蔬菜和参与捣乱的绿棘球，玩家需要模拟 202020 轮（一个季节 555 轮）的种植过程。每轮会随机给出两种种植模式，玩家必须选择其中一种种植上去（若无空地可种会扣 555 分），这被称为 事件。游戏结束时根据四种结算规则计算总得分：

  • 每个季节/每个关卡结束时根据当前种植情况检查某种要求是否达成来奖励，被称为 季节结算 和 土地结算。
  • 游戏结束时统计洒水器周围的植物数量作为奖励，被称为 科技结算 。
  • 游戏结束时统计绿棘球周围的空地数量作为惩罚，被称为 绿棘结算。绿棘球在某些轮会随机撒在空地上。

个人攻略和感想：技术为主运气为辅。我根本没有心思去研究所有的事件模组和结算模组，直接硬着头皮去种。主要的宗旨是花式填满空地来完成尽量多的种植要求（最多跳过一两轮），否则分数会很难看。我会有意识地留一些较大的方块，为冬季过分的要求做准备。然后就是要尽量贴进季节结算和土地结算的要求。

2022-3-18 风来踏雾

游戏规则介绍：从一条路的左侧出发不回头地向右走，到终点后能再不回头地向左走一趟。每一个格子上都有 333 个商店，分别花 3,2,13,2,13,2,1 金币能购买指定的物资组合，购买后会立即刷新物资。某个商店多次刷新后会变成“迷雾”，无法继续购买。背包总共有 999 格子（最多放 999 个物资组合）。同一时刻有 333 份备选订单，完成某份订单后会获得对应的分数并刷新这份订单，订单要求的物资数量越多分数就越多。游戏一共有 202020 个回合，每回合最多能操作 444 次，每次可以做 444 种事：获得一单位金币；购买一份物资组合；前进一步；提交一份订单。要求回合结束后的金币尽量多。一共有三个初始人物可供选择，每种人物都有额外的 buff：风之子岚（每次移动到新区域都能获得 2 金币），琉彩（每次购买精品区都能获得 111 的分数），塔威尔（每完成一个任务后都能获得一份物资）。

个人攻略和感想：风之子岚最中规中矩，能获得保底的 2×6=122 \times 6=122×6=12 个金币。塔威尔的可操作性很强，但是很容易超物资上限，多出来的商品很容易打乱原本的订单计划。琉彩相当于是锦上添花。我发现次数容易吃紧（最多只能操作 20×4=8020 \times 4=8020×4=80 次），所以常常会沿路直接把商店掏空，从而节省下回程的步数——当然这需要有强大的体力控制的能力。订单的话优先刷容易达成的；初期的大订单可能会卡你好久，建议咬咬牙先做掉。

2022-3-25 遗迹修复

游戏规则介绍：有 333 种物资（石头，木头，砖块），每种分为 1,2,3,41,2,3,41,2,3,4 四个等级，正好对应 3×43 \times 43×4 种卡牌。每种等级的物资都对应一个库存（界面右侧），初始为都 000。有一个大小为 999 的卡槽（界面下方），初始时会给出等级为 111 的每种物资各一张。每一轮，玩家可以选择一批等级相同的卡片“使用”，规则如下：

  • 使用后卡牌消失，转化成库存里真正的物资数量（一张卡牌正好能增加对应物资的一点数量）；
  • 若使用卡牌的等级为 xxx，还会补充 xxx 张随机卡牌（若超过卡槽上限则无法使用）。
  • 当卡槽中某种物资存在多等级的卡牌时，只能使用最高等级的卡牌。

游戏有一些固定的任务（界面上方），每当库存物资数量达到任务的物资量时可以提交该任务，获得对应的分数并扣除相应库存。保证先从高等级的物资扣起。当无法使用卡牌时游戏结束，使结束后的分数最大。

个人攻略和感想：最喜欢的一个游戏。看上去随机性很大，其实策略占了绝对主导地位，很容易完成大部分任务。最理想情况是卡槽里全是等级为 111，每次使用一张就能加一个对应库存并新增一张，相当于永动机。高等级的卡片会让局面特别被动——它会阻止你使用同种物资的低等级卡片，且容易卡满卡槽。我的主要策略如下：

  • 优先使用等级低的卡牌。比如我会一直用等级为 111 的卡牌直到卡槽里所有等级都 ≥2\ge 2≥2。
  • 高等级的卡牌尽量全合在一起使用以减少新卡牌的生成，除非此时卡槽特别空。
  • 卡槽中的卡牌数量保持在较低的水平，最多别超过 666，确保能正常用掉一次等级为 444 的卡牌。
  • 优先完成所需物资数量多、种类丰富的任务。

2022-4-15 杰洛特的营救计划

游戏规则介绍：在一个含有障碍物的网格里去捕捉一只精灵（重合算捉住），每回合能上下左右移动一步，精灵在每回合也可能随机移动一步。精灵默认不可见，战斗胜利后可兑换呼叶叶之风来查看当前精灵的位置。此外还有两种道具：1. 冻结装置：固定在某一个格子上，精灵进入后会直接被捉住，最多放 222 个；2. 电网屏障：固定在某一个格子上，使用后精灵无法进入或跨过这个格子，最多放 555 个。

个人攻略和感想：难度不高，但如何利用道具最快速度抓住它是个学问。

2022-5-2 赛尔雾中行-修复电路

游戏规则介绍：给出三列电路元件，每一列是 666 种颜色的排列，每行相邻两列有电路。一开始所有颜色都是未知的，要求点亮所有颜色。每次可以交换同一列的两个元件，若交换后某行相邻元件颜色相同，该元件就会被点亮。

个人攻略和感想：只要不去点击那些已经被点亮的元件，整个图就只会从暗变亮。可能是因为排列数量不够大，不断地无脑点击任意两个同列的暗色元件就能很快地点亮全图。

2022-5-2 赛尔雾中行-龙骨回响

游戏规则介绍：给出一个包含上下左右箭头的网格图。每次可以交换两个格子的箭头，要求能从左上角走到右下角。当前的行走路径会被高亮出来。

个人攻略和感想：把周围的向右和向下箭头慢慢凑成一条路。感觉这个游戏也能出题？

2022-7-20 游乐园设施修复

游戏规则介绍：每一个格子里的管道可以旋转，要连通电线头到电线尾。

个人攻略和感想：难度不大但是很繁琐。根据这个游戏的 idea 出了道 牛客多校题，本来想出更难的策略类的题失败了，变成了 BFS 模拟题。我还找到了一个拓展后的游戏：https://hexapipes.vercel.app/hexagonal。
