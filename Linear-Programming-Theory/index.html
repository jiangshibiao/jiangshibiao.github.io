凸集、凸函数、凸优化

凸集：任取 x,y∈Sx, y \in Sx,y∈S 和 ∀θ∈[0,1]\forall \theta \in [0,1]∀θ∈[0,1] 满足 θx+(1−θ)y∈S\theta x + (1-\theta)y \in Sθx+(1−θ)y∈S，称集合 SSS 凸集（Convex Set）。

  • 凸集的交仍然是凸集。
  • 如果没有 ∀θ∈[0,1]\forall \theta \in [0,1]∀θ∈[0,1] 的条件，称集合 SSS 仿射集（Affine set）。

凸函数：对于定义在凸集 SSS 上的函数 f(x)f(x)f(x)，若对于 ∀θ∈[0,1]\forall \theta \in [0,1]∀θ∈[0,1] 有 f(θx+(1−θ)y)≤θf(x)+(1−θ)f(y)f(\theta x + (1-\theta) y) \le \theta f(x) + (1-\theta) f(y)f(θx+(1−θ)y)≤θf(x)+(1−θ)f(y)，那么称 f(x)f(x)f(x) 是凸函数（Convex Function）。

  • 延森不等式（Jensen’s inequality）：若 f(x)f(x)f(x) 是凸函数， f(∑i=1nθixi)≤∑i=1nθif(xi)f(\sum_{i=1}^n \theta_ix_i) \le \sum_{i=1}^n \theta_i f(x_i)f(∑i=1n​θi​xi​)≤∑i=1n​θi​f(xi​)
  • 定理：凸函数的局部最优就是全局最优。
    1. 假设 x^\hat{x}x^ 是局部最优点，x∗x^\astx∗ 为全局最优点，则 f(x^)>f(x∗)f(\hat{x}) > f(x^\ast)f(x^)>f(x∗)。

  • 凸函数的判定条件：

线性规划的定义

线性规划定义：

  • 左边形式的问题被称为线性规划（Linear Programming）
  • 由于仿射函数既是凸函数又是凹函数，所以优化问题是 min 还是 max 问题不大；常数 ddd 对优化问题的解没有影响，一般也可以去掉。这就变成了中间的这个优化函数。
  • 对于 Ax≤bAx \le bAx≤b 这个约束，可以通过添加非负变量将其松弛成等式。所以我们总能将线性规划转成右侧的形式。右式也称为线性规划的典则形式。

极点：若 x∈Sx \in Sx∈S 无法表示为凸集 SSS 内某两个元素的凸组合，称 xxx 是极点（extreme point）。

  • LP 问题的可行域实际上是很多超平面的交，最后组成的应该是一个超多面体。
  • 极点就是超多面体的“顶点”。可以证明，若最优解有解，必然取在极点上：
    1. 假设最优点 OOO 严格在凸集里面。任取一条与凸集交在 PPP 和 QQQ 的直线。因为是线性规划问题，OOO 点处的函数值一定是 PPP 点和 QQQ 点的线性组合。所以 PPP 和 QQQ 中，至少有一处的函数值是大于等于 OOO 的。
    2. 我们可以重复上述步骤，把最优点一步一步约束到顶点上。比如在三维中，我们先将其“规约”到凸壳的表面上，最后“规约”到凸壳的边和点上。

基可行解：我们讨论 Ax=bAx = bAx=b 有解且行满秩的情况（如果不满秩就去掉线性相关的限制条件）。设 AAA 是一个 m×nm \times nm×n 的矩阵，根据线性代数的知识，我们可以从 AAA 中选出最多 mmm 列线性无关的列向量，其它列向量都和它们线性相关。我们把这 mmm 个列向量调整到前面去，把 AAA 分成两部分：A=[ABAN]A = \begin{bmatrix} A_B & A_N \end{bmatrix}A=[AB​​AN​​].容易构造出 Ax=bAx = bAx=b 的一个解：x=[AB−1b0]Tx = \begin{bmatrix} A_B^{-1}b \quad 0 \end{bmatrix}^Tx=[AB−1​b0​]T，称这种解为基可行解（Basic Feasible Solution）。显然，基可行解至多有 CnmC_n^mCnm​ 种。

  • 定理1：每个极点都对应着一个基可行解，且每个基可行解都对应着一个极点。
  • 定理2：最优解一定可以在基可行解处取到。

单纯形法

单纯形法（Simplex Method）（以 max⁡\maxmax 模型为例）

  1. 将 AAA 中的基变量消成单位矩阵，并把目标函数基变量的系数都消成 000。
  2. 以某种策略找一个 cj>0c_j>0cj​>0 的非基变量 jjj 入基。
  3. 依次考察 bi>0b_i > 0bi​>0 的行，确定一行 iii 使得 biAi,j\frac{b_i}{A_{i,j}}Ai,j​bi​​ 最小（对于 jjj 最紧的限制）。
  4. 拿第 iii 行去消，把每一行的 A?,jA_{?,j}A?,j​ 以及目标函数的 cjc_jcj​ 都消成 000。即第 iii 行的基变量出基。
  5. 重复步骤 2∼42 \sim 42∼4 直到检验数全大于 000.

评价

  • 变换的本质是：在满足约束的限制下，用不同的非基变量组合来表示目标函数。
  • 当非基变量的检验值均 ≤0\le 0≤0 时取到最大值。我们可以想象成，目标函数就是由这些非基变量决定的，它们目前都取了 000，而且任意变量的微小增长都会导致结果变劣。
  • 单纯形正确性证明
    1. 设最终的检验数是 c^\hat{c}c^（均小于 000），对应的解是 xxx，最优解是 yyy。
    2. 令 d=x−yd=x-yd=x−y，我们有 Ax−Ay=b−b=0=Ad=ABdB+ANdNAx - Ay = b - b = 0 = Ad = A_Bd_B + A_Nd_NAx−Ay=b−b=0=Ad=AB​dB​+AN​dN​

    4. 我们知道 xN=0x_N = 0xN​=0，又 y≥0y \ge 0y≥0，所以 dN≥0d_N \ge 0dN​≥0；又 c^≤0\hat{c} \le 0c^≤0，所以 cTd=c^dN≤0c^Td = \hat{c}d_N \le 0cTd=c^dN​≤0，那么 cTy=cTx+cTd≤cTxc^Ty = c^Tx + c^Td \le c^TxcTy=cTx+cTd≤cTx，说明 yyy 并没有比 xxx 更优。
  • 在非退化情况下，单纯形法一定可以停止。因为每次迭代都会让答案更优一点，访问的基可行解空间是不会重复的。而基可行解数量又是有限的，所以算法一定可以停止。
  • 可以证明，如果按字典序的顺序选取 jjj，单纯形算法一定可以停止。

tsr 讲解单纯形法

单纯形法的初始可行解

初始可行解

  • 若单纯形的约束是 Ax≤bAx \leq bAx≤b，初始可行解可以取 x=0x = \pmb{0}x=0.
  • 若约束是 Ax=bAx = bAx=b 时，我们可以通过加入松弛变量 xˉ\bar{x}xˉ 使得 Ax+xˉ=bAx + \bar{x} = bAx+xˉ=b，这样 x=0,xˉ=bx = \pmb{0}, \bar{x} = bx=0,xˉ=b 是初始可行解。这里存在一个问题：xˉ\bar{x}xˉ 是我们添加进去的变量——我们希望 xˉ\bar{x}xˉ 能全部出基。

大 M 法（Big M Method）

  • 对不为 000 的 xˉ\bar{x}xˉ 进行“惩罚”。将目标函数改为 z=cTx−M∑i=1mxˉiz = c^Tx - M\sum_{i=1}^m\bar{x}_iz=cTx−M∑i=1m​xˉi​。如果 MMM 是一个足够大的正数，xˉ\bar{x}xˉ 就会在 −M-M−M 这个“严厉的惩罚”之下变成 000。
  • MMM 取多大很难说，且取得过大会影响精度。

两阶段法（Two-Phase Method）

  • 添加了 xˉ\bar{x}xˉ 后，我们考虑一个新的线性规划：约束不变，改成优化 min⁡∑xˉi\min {\sum \bar{x}_i}min∑xˉi​ 这个函数。一组显然的初始可行解是 x=0,xˉ=bx = \pmb{0}, \bar{x} = bx=0,xˉ=b.
  • 如果这个优化问题的最优解的目标函数值不为 000，则原问题无可行解；否则我们就找到了原问题的一组可行解。我们再以这个可行解为起点，利用单纯形法求出原问题的最优解即可。
  • 由此可以发现，线性规划的可行解和最优解求解难度相同。

思考：对于 max⁡cTx,Ax≤b\max c^Tx,Ax \le bmaxcTx,Ax≤b 这个模型，如果突然多了一个 x1≥4x_1 \ge 4x1​≥4 的条件怎么办？

  • 添加一个松弛变量 x‾\overline{x}x： −x1+x‾=−4-x_1+\overline{x} = -4−x1​+x=−4
  • 此时会发生一个问题：x‾\overline{x}x 和 bbb 至少有一个会取负号，不符合约束。
  • 怎么办呢？再引进一个变量 x^\hat{x}x^：x1−x‾+x^=4x_1-\overline{x}+\hat{x}=4x1​−x+x^=4
  • 至此，x^=4\hat{x}=4x^=4 是新问题的一个可行解，直接套大M法或二阶段法求解。

对偶定理与对偶单纯形法

线性规划的对偶

对偶定理

  • 弱对偶定理 (Weak Duality)：设 xxx 和 yyy 分别是原问题和对偶问题的可行解，则 cTx≤bTyc^{\rm T}x \le b^{\rm T}ycTx≤bTy。证明：由 yyy 的可行性我们有 ATy≥cA^{\rm T}y \ge cATy≥c，即 yTA≥cTy^{\rm T}A \ge c^{\rm T}yTA≥cT，两边同乘以 xxx 有 yTAx≥cTxy^{\rm T}Ax \ge c^{\rm T}xyTAx≥cTx；由 xxx 的可行性还有 Ax≤bAx \le bAx≤b，那么 yTAx≤yTby^{\rm T}Ax \le y^{\rm T}byTAx≤yTb，合起来就是 cTx≤bTyc^{\rm T}x \le b^{\rm T}ycTx≤bTy。
    • 最优性：若 xxx 和 yyy 分别是原问题和对偶问题的可行解，而且 cTx=bTyc^\mathrm{T}x=b^\mathrm{T}ycTx=bTy，那么 xxx 和 yyy 分别是原问题和对偶问题的最优解。
    • 可行性：若原问题无最优解（可以取无穷大），则对偶问题无可行解；若对偶问题无最优解（可以去无穷小），则原问题无可行解。
  • 强对偶定理 (Strong Duality)：若原问题（或对偶问题）有有限最优解，那么对偶问题（或原问题）也有有限最优解，且二者最优解相等。通过单纯形法的计算过程来辅助证明。
  • 互补松弛定理 (Complementary Slackness)：若 x∗x^\astx∗ 与 y∗y^\asty∗ 分别是原问题和对偶问题的可行解，以下两点等价：
    1. x∗x^\astx∗ 和 y∗y^\asty∗ 分别是原问题和对偶问题的最优解；
    2. (y∗TA−cT)x∗=0(y^{\ast {\rm T}}A - c^{\rm T})x^\ast = 0(y∗TA−cT)x∗=0 且 y∗T(Ax∗−b)=0y^{\ast {\rm T}}(Ax^\ast-b) = 0y∗T(Ax∗−b)=0。

对偶单纯形法步骤（以 min⁡\minmin 模型为例）

  1. 取一组检验数全都 ≥0\ge 0≥0 的基。
  2. 找一个 bi<0b_i < 0bi​<0 （一般找绝对值最大的）行 iii，将非基变量 iii 出基。
  3. 我们想让某个非基变量从 000 变成大于 000 来增加 bib_ibi​。在这一行中，找一个 Ai,j<0A_{i,j}< 0Ai,j​<0 的 jjj 使得 ∣cjAi,j∣|\frac{c_j}{A_{i,j}}|∣Ai,j​cj​​∣ 最小（这样 jjj 消完目标函数后检验数依然全都 ≥0\ge 0≥0），让 jjj 入基。
  4. 重复步骤 2∼32 \sim 32∼3 直到约束右侧的值均 ≥0\ge 0≥0 。
