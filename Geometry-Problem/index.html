  • AAAI 2014 论文地址
  • 本文提出了一个叫做 G-ALIGNER 的 Image Parse 的工具，并证明了它使用的估价函数是次模函数。
  • 估价函数介绍
    • 设 T={T1,T2,…,Tm}T=\{T_1,T_2,\dots,T_m\}T={T1​,T2​,…,Tm​} 表示题目文本中提到的元素集合，LLL 是图片中初步提取后的所有 primitives 集合。本论文默认 LLL 已经得到，现在要找一个最优的子集 L^\hat{L}L^ 以准确地表达图片信息。
    • 引入一个 W∈{0,1}∣L∣×∣T∣W \in \{0,1\}^{|L| \times |T|}W∈{0,1}∣L∣×∣T∣ 表示该图上的 primitive 是否与指定文本元素对应。 L^=L×(W×1∣T∣×1)\hat{L}=L \times (W \times 1_{|T| \times 1})L^=L×(W×1∣T∣×1​)，可以将求一个最优子集 L^\hat{L}L^ 转化为求一个最优的矩阵 W^\hat{W}W^。
    • 对于一个可能的集合 L^\hat LL^，估价函数 F\mathcal{F}F 设计成三个部分：
      • 所选 primitives 与实际的图像之间的重合程度 P=∣DL^∣∣D∣\mathcal{P}=\frac{|D_{\hat L}|}{|D|}P=∣D∣∣DL^​∣​
      • 所选 primitives 能构成更高相关性 visual element 的程度 C=∣CL^∣∣C∣\mathcal{C}=\frac{|C_{\hat L}|}{|C|}C=∣C∣∣CL^​∣​
      • 所选 primitives 与文本的对应关系 S=∣TL^∣∣T∣−rW^\mathcal{S}=\frac{|T_{\hat L}|}{|T|}-r_{\hat W}S=∣T∣∣TL^​∣​−rW^​，其中 rW^r_{\hat W}rW^​ 表示冗余成分。
    • 可以证明，F=P+C+S\mathcal{F=P+C+S}F=P+C+S 是次模函数。

    • 用 Hough transform 提取 primitives，用 Harris Corner detectors 检测 Corners。

A Generalized Earley Parser for Human Activity Parsing and Prediction

  • This paper is an extension of previous ICCV and ICML papers. 1TPAMI 2020 Address

  • Earley Parser is a classical method to parse CFG, while it can not tackle the non-Markovian and compositional properties of high-level semantics. This paper generalize the Earley parser to solve sequence data by only given an arbitrary probabilistic classifier.

  • This Generalized Earley Parser combined detection, parsing and future predictions together, which is really generic and applicable. It finds the optimal (maximizing the joint probability) segmentation and label sentence according to both a symbolic grammar and a classifier output of probabilities of labels for each frame.

  • Brief introduction for EARLEY PARSER

    • Early parser takes a sentence with length TTT as input, and find out this CFG.
    • It is similar with LR parser. There are three main process:
      • Prediction: For each (A→α⋅Bβ)(A \to \alpha \cdot B\beta)(A→α⋅Bβ), add (B→⋅γ)(B \to \cdot \gamma)(B→⋅γ) if B→γB \to \gammaB→γ exists.
      • Scanning: If (A→α⋅aβ)(A \to \alpha \cdot a \beta)(A→α⋅aβ) and the next character is α\alphaα, add (A→αa⋅β)(A \to \alpha a \cdot \beta)(A→αa⋅β).
      • Completion: If (A→γ⋅)(A \to \gamma \cdot)(A→γ⋅) appears, change (B→α⋅Aβ)(B \to \alpha \cdot A\beta)(B→α⋅Aβ) to (B→αA⋅β)(B \to \alpha A \cdot \beta)(B→αA⋅β).

Closed Loop Neural-Symbolic Learning via Integrating Neural Perception, Grammar Parsing, and Symbolic Reasoning

  • ICML2020. Propose a Neural-Grammar-Symbolic Model (NGS).
  • Learning neural-symbolic models with RL approach usually converges slowly with sparse rewards. The author proposes a novel top-down back-search, which can also serve as Metropolis-Hastings sampler (a sampling method in MCMC).
  • Model Inference
    • Let xxx be the input (e.g.an image or question), zzz be the hidden symbolic representation, and yyy be the desired output inferred by zzz.
    1. Neural Perception: Map xxx to a normalized probability distribution of the hidden symbolic representation zzz: pθ(z∣x)=softmax(ϕθ(z,x))p_θ(z|x) = softmax(\phi_θ(z, x))pθ​(z∣x)=softmax(ϕθ​(z,x))
    2. Grammar Parsing: Find the most probable zzz that can be accepted by the grammar: z^=arg⁡max⁡z∈L(G)pθ(z∣x)\hat z = \arg \max \limits_{z \in L(G)} p_θ(z|x)z^=argz∈L(G)max​pθ​(z∣x). We use Generalized Early Parser doing the grammar parsing.
    3. Symbolic Reasoning: Given the parsed symbolic representation z^\hat zz^, performs deterministic inference with z^\hat zz^ and the domain-specific knowledge Δ\DeltaΔ to predict yyy: y^=f(z^;∆)\hat y = f(\hat z; ∆)y^​=f(z^;∆)
  • Learning Details

    • 1-STEP BACK-SEARCH

      • Previous methods using policy gradient to learn the model discard all the samples with zero reward, but 1-STEP BACK-SEARCH can correct wrong samples and use the corrections as pseudo labels for training.
      • In the 1-BS, assume that only one symbol can be replaced at a time. Given (z^,S,y)(\hat z,S,y)(z^,S,y), the 1-BS gradually searches down the reasoning tree starting from the root node SSS to the leaf nodes and wants to find the most probable correction z∗z ^{\ast}z∗ at one-step distance.
      • During this process, maintain a priority queue storing 3-tuple (A,αA,p)(A, \alpha_A, p)(A,αA​,p): current node, current expected value (If the value of AAA changes to αA\alpha_AαA​, z^\hat zz^ will execute to the ground-truth answer yyy) and the visiting priority (probability).
      • When transformation, enumerate each child BBB of AAA and calculate expected value αB\alpha_BαB​ such that f(z^(B→αB);Δ))=f(z^(A→αA);Δ))=yf(\hat z(B \to \alpha_B); \Delta)) = f(\hat z(A \to \alpha_A); \Delta)) = yf(z^(B→αB​);Δ))=f(z^(A→αA​);Δ))=y.       • p\left(A \rightarrow \alpha_{A}\right)=\left\{\begin{array}{ll} \frac{1-p(A)}{p(A)}, & \text { if } A \notin \Sigma \\ \frac{p\left(\alpha_{A}\right)}{p(A)}, & \text { if } A \in \Sigma \and \alpha_{A} \in \Sigma \end{array}\right.
