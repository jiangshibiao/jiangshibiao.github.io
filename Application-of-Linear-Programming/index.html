原始对偶方法

分枝定界法

  • 分枝定界法的思想和最优性剪枝或者 min-max 搜索树类似。

  • 我们先解实数线性规划，得到了整数规划最优解的上界。假设最优解中 xi(k<xi<k+1)x_i(k < x_i < k+1)xi​(k<xi​<k+1) 不是整数，就会有两种可能：xi≤kx_i \le kxi​≤k 或 xi≥k+1x_i \ge k+1xi​≥k+1，对两种情况分别进行搜索。

  • 如果在某一枝内算出了一个整数解，我们就得到了原整数规划最优解的下界。

  • 某个搜索分支结束有三种可能：

    • 当前线性规划无可行解（死枝）。
    • 当前（实数）最优解未超过下界（枯枝）。
    • 当前已经是整数解（活枝）。

  • 显然，x′\pmb{x'}x′ 也是符合该线性规划的一组解（对于任意一组限制，至少存在一个 xi≥0.5x_i \ge 0.5xi​≥0.5；这时候对应的 xi′x'_ixi′​ 取成 111，满足限制要求）

Unrelated Parallel Machine Scheduling 问题

  • tsr 的 UPMS问题讲解
  • 有 mmm 台机器和 nnn 件物品，每件物品都要在一台机器上加工，第 iii 台机器加工第 jjj 件物品的时间为 ai,ja_{i, j}ai,j​。求一种把物品分配给机器的方案，使得加工总时长最长的机器加工时间最短。
  • 优化模型很简单：
    • min⁡x,tt\min\limits_{x, t} tx,tmin​t
    • s.t.∑i=1mxi,j=1∀j∈{1,2,…,n}\text{s.t.} \quad \sum\limits_{i=1}^m x_{i, j} = 1 \quad \forall j \in \{1, 2, \dots, n\}s.t.i=1∑m​xi,j​=1∀j∈{1,2,…,n}
      ∑j=1nai,jxi,j≤t∀i∈{1,2,…,m}\qquad \sum\limits_{j=1}^n a_{i, j}x_{i, j} \le t \quad \forall i \in \{1, 2, \dots, m\}j=1∑n​ai,j​xi,j​≤t∀i∈{1,2,…,m}
    • xi,j∈{0,1}x_{i, j} \in \{0, 1\}xi,j​∈{0,1}
  • 注意到 ttt 有二分性。现在我们二分 ttt，每次直接解实数线性规划来判是否存在可行解。设最终稳定的值是 TTT，对应的实数解是 x∗x^\astx∗。显然 TTT 是答案的一个下界，即 T≤OPTT \le \text{OPT}T≤OPT.
    • 对于第 jjj 件物品，若 ∃i xi,j∗=1\exists i ~ x^\ast_{i, j} = 1∃i xi,j∗​=1 那么称该物品为“整数物品”，否则称为“分数物品”。设共有 n1n_1n1​ 个“整数物品”，n2n_2n2​ 个“分数物品”，显然有 n1+n2=nn_1 + n_2 = nn1​+n2​=n
    • 若第 jjj 件物品为“分数物品”，那么 xi,jx_{i, j}xi,j​ 中至少有两个非零值。原问题有 n+mn+mn+m 条限制，根据单纯形法，非零的变量至多有 n+mn+mn+m 个，那么 n1+2n2≤n+mn_1 + 2n_2 \le n + mn1​+2n2​≤n+m，得到 n2≤mn_2 \le mn2​≤m。
  • 下面我们给每台机器分配物品，使得每台机器的加工总时长都不超过 2T2T2T，以此说明近似比是 222。
    • 先做一个小处理：把用时大于 TTT 的物品都移除（显然是等价的）。
    • 构造一张二分图：左边有 mmm 个点，每个点表示一台机器；右边有 nnn 个点，每个点表示一件物品。若 xi,j>0x_{i, j} > 0xi,j​>0 则连接第 iii 台机器和第 jjj 件物品。
    • 一个重要的性质：该二分图边数不超过点数。
    • 首先考虑“整数物品”。很显然，每个“整数物品” jjj 都只和一台机器 iii 有连边 (i,j)(i, j)(i,j)，那就把“整数物品” jjj 放在机器 iii 中加工，并去掉物品 jjj 和边 (i,j)(i, j)(i,j)。由于每次恰好去除一个点和一条边，性质依然成立。
    • 这样，图中就只剩机器和“分数物品”了。考虑所有度数为 111 的机器 iii。假设唯一连接机器 iii 的边是 (i,j)(i, j)(i,j)，那么我们把“分数物品” jjj 放在机器 iii 中加工，并去掉机器 iii、物品 jjj 和物品 jjj 的所有连边。由于每件“分数物品”都有至少两条连边（度数至少为 2），所以我们每次都会去掉两个点以及至少两条边，性质依然成立。
    • 反复操作直到最后不存在度数为 111 的机器为止，剩下的图一定是由若干个偶环构成。所以从始至终，我们总能通过合理分配，使得每台机器至多分配到一个“分数物品”。每台机器在完成原来若干整数任务的同时，独立地把分配给自己的那个“分数物品”做完。注意一件物品的加工时间至多为 TTT，所以现在的用时最多是 2T2T2T。
